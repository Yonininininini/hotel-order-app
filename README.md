# Hotel Finance & Waiter Order Management App

A simple web application for hotels to manage waiter orders, kitchen status, and generate bills.

Built with:
- Python
- Flask
- SQLite (for database)
- HTML/CSS (Bootstrap)
- WeasyPrint (PDF generation)
